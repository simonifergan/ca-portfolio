Instructions

Welcome to a Minesweeper-esque game by Simon Ron Ifergan. ©

About this version:

  • The first cell you open will never be a mine.
  • In higher difficulties (Medium & Expert) all mines must be flagged before you can win the game.
  • Game starts in Medium difficulty. You may change it anytime. Note that changing the difficulty in the middle of the game restarts the current run.
  • You have 3 hint buttons to use per run. Each button will fade upon use.
  • Don't be like me, I can't win at this game without using hints.
  • Your total time will be your score at the end of each run.
  • Click on the face next to the timer if you wish to restart the game.
Keybinds:
  • Left Mouse Button - Open Cell
  • Right Mouse Button - Flag/Unflag Cell
Enjoy the game,
Simon.
